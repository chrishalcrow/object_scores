# object_scores

## Installation

In a terminal, make a virtual environment and activate it. For this example, I'll call it `object`

```
python -m venv object
source activate object 
```

Now clone (download) this repo, move inside the folder you download and install the package:

```
git clone https://github.com/chrishalcrow/object_scores.git
cd object_scores
pip install -e .
```
